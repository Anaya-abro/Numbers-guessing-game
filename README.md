# Number Guessing Game

A simple and beginner-friendly Python game where you try to guess a number between 1 and 100.

---

## Features

- Three difficulty levels:
  - Easy (15 lives)
  - Normal (10 lives)
  - Hard (5 lives)
-  Random number generation
-  Feedback on guesses (Too High / Too Low)
- Replay option after each game
- Tracks number of attempts

---

## How to Play

1. Run the Python file
2. Choose a difficulty level (1, 2, or 3)
3. Guess a number between 1 and 100
4. The game will tell you:
   - "Too High" 
   - "Too Low" 
5. Keep guessing until:
   - You win 🎉
   - Or you run out of lives 
6. Choose to play again or exit

---

##  How to Run

Make sure you have Python installed, then run:
